a target value or range of values for a service level that is measured by an SLI. -- Google SRE Book

Service Definition

Service Level Objectives

Examples:
  • Request latency:
    • 50th percentile ≤ 50 milliseconds
    • 99th percentile ≤ 2 seconds
  • Availability:
    • 99.0% (2 nines)
    • 99.5%
    • 99.999% (3 nines)
    • N nines
